# Pago Empleados

### Utiliza swing 

Proyecto realizado en Java en el cual se procesa el pago de empleados.

#### Calculos

    El salario basico    = Horas*Tarifa
    La bonificacion:    = *0.2                                        (20%)
    El neto                   =Basico+Bonificacion)*0.9     (90%)
